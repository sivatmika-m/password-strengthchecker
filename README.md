# Password Strength Checker

## Objective

The Password Strength Checker is a Python program that evaluates the strength of a password based on various security factors.

## Features

* Checks password length
* Verifies uppercase letters
* Verifies lowercase letters
* Checks for numeric digits
* Checks for special characters
* Classifies passwords as Weak, Medium, or Strong
* Provides feedback to improve password security

## Technologies Used

* Python 3

## How It Works

1. The user enters a password.
2. The program checks:

   * Minimum length (8 characters)
   * Presence of uppercase letters
   * Presence of lowercase letters
   * Presence of numbers
   * Presence of special characters
3. Based on these checks, the password is classified as:

   * Weak Password
   * Medium Password
   * Strong Password
